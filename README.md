# AirAI

Welcome to AirAI - a secure AI chat platform with member authentication.

## What Is This?

AirAI is a web-based chat application that allows users to interact with AI models. Access is controlled through a secure login system with time-limited session codes.

## How To Use

### 1. Visit the Website
Go to the published site and you'll see the login page.

### 2. Choose Your Login Type

**Member Login**
- Provides 15-minute access
- Session code expires after time limit
- Ideal for temporary access

**Guest Login**
- Unlimited access time
- No expiration on session code
- Perfect for extended sessions

### 3. Get Your Access Code
After logging in, you'll receive a unique member key. This is your personal access code - keep it secure!

### 4. Enter the Chat
Click "Next: Open Chat Interface" to access the AI chat platform with your member key.

### 5. Start Chatting
Once inside, you can start conversations with the AI models available.

## Frequently Asked Questions

**Q: What if my session expires?**
A: Simply log in again to get a new member key.

**Q: Can I use the same member key twice?**
A: No, each login generates a new unique key.

**Q: What's the difference between Member and Guest?**
A: Members have 15-minute time limits, Guests have unlimited access.

**Q: Is my data secure?**
A: Yes, all sessions are managed securely and member keys are unique.

**Q: Can I access this on my phone?**
A: Yes! The interface is fully responsive and works on all devices.

**Q: Do I need to create an account?**
A: No registration required - just choose Member or Guest login.

## Support

Having issues? Common solutions:

- **Login not working**: Check your internet connection and try again
- **Chat won't load**: Make sure you entered the member key correctly
- **Session expired**: Log in again to get a new access code
- **Page not loading**: Clear your browser cache and refresh
